# page-loaded-time

This module will return how much seconds does it take for your site to load.

# Installation
`$ npm install page-loaded-time --save`

# Usage
```javascript
import pageLoadedTime from "page-loaded-time";

const time = pageLoadedTime();

console.log(time); // 0.932s
```
